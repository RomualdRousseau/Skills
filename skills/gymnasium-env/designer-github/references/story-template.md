# User Story Template

Use this template when creating new features or mechanics.

## Format
**Story**: As a [role], I want to [action], so that [value/outcome].

### Acceptance Criteria
- [ ] Criterion 1
- [ ] Criterion 2
- [ ] Criterion 3

### Technical Notes (Optional)
- [ ] Implement [logic] in `src/game/[module].py`
- [ ] Add [asset] to `src/assets/`
