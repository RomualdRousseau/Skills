# Slide Generator Design System

The mandatory visual and technical standard for generating high-fidelity technical slides via HTML5 and Tailwind CSS.

## Technical Stack
- **Framework:** Tailwind CSS.
- **Icons:** Lucide Icons.
- **Fonts:** 'Inter' (Primary), 'JetBrains Mono' (Technical).

## Visual Identity
- **Base Color:** Slate (`slate-50` to `slate-900`).
- **Primary Accent:** Indigo-600.
- **Secondary Accents:** Emerald (Success), Purple (Logic), Amber (Warning), Rose (Error).
- **Layout:** 16:9 aspect ratio, centered container, rounded-xl/2xl corners, shadow-2xl.

## UI Components
- **Technical Cards:** Grouped content with mandatory `.card-hover` effects (translate-y-4 + shadow).
- **Icon Containers:** Wrapped in rounded squares with background accents (e.g., `bg-indigo-50`).
- **Technical Badges:** Small font-mono tags for categorizing technical terms.
